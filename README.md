# Hamza Maqbool - Personal Portfolio 🚀

A clean, responsive, and interactive personal portfolio website built with React.js to showcase my skills, experiences, projects, and achievements.

## ✨ Features
- Custom interactive UI with Lottie animations
- Responsive design for all devices (Mobile and Desktop)
- Sections for Skills, Work Experience, Education, Projects, and Blogs
- Light / Dark Mode support

## 🛠️ Built With
- **React.js**
- **Sass / SCSS** for styling

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) and `npm` installed.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hamza-mqbl/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
